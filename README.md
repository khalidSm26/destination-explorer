# Travel-destination-guide
A responsive 5-page travel website showcasing tourist cities through images and descriptions. Each page presents a different destination with curated visuals and structured content, designed to provide an engaging and user-friendly browsing experience.

## Features
- Developed a 5-page website showcasing different tourist destinations  
- Integrated visual content (images) with informative descriptions  
- Designed a user-friendly navigation system across all pages  
- Implemented responsive design for optimal viewing on various devices  
- Organized content using modern layout and styling techniques

  ## Tech Stack
- HTML
- CSS
- JavaScript
